# Counting Sheep
MSCH-C220 Final Project

## Theme
I took the theme more literally than thematically and made a game with only one button/control. When people think of only having one button in a game they might assume its boring so I leaned into that by making my game put you to sleep.

## Game Play
The idea is that you are a sheep jumping over infinite fences to make the player fall asleep. There's no real end to the game unless you die (hit the fence) and there is a timer to see how long you can play. The controls are space for jump and you must jump over the fences or else you will "die".

## What did you 
I had trouble making the animation for the sheep and I tried making a more realistic walk cycle first but in the end decided to go for something way simpler. I think it fits better overall. I referenced a tutorial to help with some of the scripting, but I couldn't follow it exactly because of some issues with the video and because it didn't fit exactly what I wanted to do. I had to change some things through trial and error a bit, but it wasn't too difficult (the biggest thing was that I had to change the respawn mechanic so I could have a menu system).

## Implementation
The art assets I tried to keep simple so I just did them myself. The sound effects are created in Chiptone and the two music tracks are from Universal Production Music--they're titled "Star Net" and "Serenading Stars". I also watched some tutorial videos from @thehelloworldguyofficial on Youtube.


## Special Focus
I guess I focused on art since I made the assets myself and did a little animation for the sheep. I also made the background looping so it looks more like you're moving. Honestly, I'd rate it a 2-3 because it was a bit simple.
